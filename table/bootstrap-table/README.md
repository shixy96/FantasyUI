## Local develop

### Init
```bash
npm i
cd demos
npm i
```

### Debug
To develop bootstrap-table locally please run:

```bash
npx http-server
```

And then open: http://localhost:8080/demos

Start server mock api

```bash
cd demos
node app.js
```

## Local build

To build bootstrap-table locally please run:

```
npm run build
```

Result will appear in `dist` directory.
